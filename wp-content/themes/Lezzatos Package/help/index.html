A) Installation - top

To install this theme you must have a working version of Wordpress already installed. You can install this theme in two ways:

  • FTP Upload: Using FTP software, first you must extract lezzatos.zip then upload all the files to /wp-content/themes/ folder on your server.
  • Wordpress Upload: Navigate to Apperance > Add New Themes > Upload page. Select lezzatos.zip file. Click Install Now button to upload and install the theme.

Once theme uploaded you need to activate it. Navigate to Apperance > Themes and activate Lezzatos theme. On first activation theme is required to reset theme options settings by navigate: Appearance > Theme Options > Options Reset.

B) Setting Up Homepage - top

After activating the theme, you'll get a blank home page that ready to be filled. Follow the instructions below:

1. Create New Slider Item

To create new slider item, navigate to: Slider > Add New.

  1. Type slider caption
  2. Type slider text
  3. Click Browse to insert picture
  4. Insert URL (optional)
  5. Click Publish. Done!
  6. Repeat step 1-4 if you wish to create as many slider as you wish

2. Create Navigation

Navigate to Appearance > Menu
  1. Type Menu Name then click Save Menu
  2. Select Primary Menu that you just created then click Save
  3. Create custom links Label "Home", set URL to you website (http://yourdomain.com) then click Add to Menu
  4. Click Save Menu
  5. Now you will see 'Home' menu on left navigation

C) Setting Up Menu - top

Menu is the main part of this theme. You can create menu item & category as many as you wish.

1. Create New Menu Item

  1. To create new portfolio item, navigate to: Menu > Add New.
  2. Type menu name
  3. Type menu description
  4. Type menu price
  5. Create/select category
  6. Click Set featured image to insert menu picture
  7. Click Publish. Done!
  8. Repeat step 1-7 if you want to create menu as many as you wish

2. Create Menu Page (List View)

To create new portfolio item, navigate to: Menu > Add New.
  1. Type page name
  2. Insert menu shortcode (type mannualy)
    [menus name="menu-category-name"]
  3. Click Publish

3. Create Menu Page (Thumbnail View)

To create new portfolio item, navigate to: Menu > Add New.
  1. Type page name
  2. Select menu thumbnail view from Page Attributes
  3. Click Publish

D) Setting Up Gallery - top

1. Setting Up Gallery

  1. Navigate to Page > Add New
  2. Type page name
  3. Select 'Gallery' template from Page Attributes
  4. Click Publish

2. Create New Gallery Item

  1. To add new gallery item, navigate to: Gallery > Add New.
  2. Type image title
  3. Type description
  4. Click Set featured image to insert picture
  5. Select Category or create new one:
  6. Click Publish. Done.

E) Setting Up Blog - top

  1. Navigate to Page > Add New
  2. Type page name
  3. Select Blog template from Page Attributes
  4. Click Publish

To add new post, navigate to: Post > Add New.

 

 

F) Setting Up Contact Page - top

  1. Navigate to Page > Add New
  2. Type page name
  3. Select Contact template from Page Attributes
  4. Click Publish
  5. Please read on Theme Option to change email destination.

G) Setting Up Reservation Page - top

  1. Navigate to Page > Add New
  2. Type page name
  3. Select Reservation template from Page Attributes
  4. Click Publish
  5. Please read on Theme Options to change email destination.

H) Setting Up Fullscreen Video - top

Navigate to Page > Add New

  1. Type page name
  2. Select template "Video Background Youtube/Video Background Vimeo" from Page Attributes
  3. Type Youtube/Vimeo video id in editor
  4. Click Publish. Done.

H) Shortcodes - top

1. Column

Description Shortcode
One half column [one_half] your content here [/one_half]
One half column last [one_half_last] your content here [/one_half_last]
One third column [one_third] your content here [/one_third]
One third column last [one_third_last] your content here [/one_third_last]
Two third column [two_third] your content here [/two_third]
Two third column last [two_third_last] your content here [/two_third_last]

2. Button

Description Shortcode
simple button [button title="Sample button" url="your url" target="_blank"]
rounded button [button title="Sample button" url="your url" target="_blank" style="rounded"]
big button (you can set button size by add attribute size="") [button title="Sample button" url="your url" target="_blank" style="rounded" size="big"]
bigger button [button title="Sample button" url="your url" target="_blank" style="rounded" size="bigger"]
biggest button [button title="Sample button" url="your url" target="_blank" style="rounded" size="biggest"]
bigger button with custom color (you can change color by replace bgcolor and textcolor) [button title="Sample button" url="your url" target="_blank" style="rounded" size="bigger" bgcolor="#00CC99" textcolor="#FFFF00"]

3. Messages Box

Description Shortcode
Success message [msgbox type="success"] Yeah message here! [/msgbox]
Warning message [msgbox type="warning"] Yeah message here! [/msgbox]
Error message [msgbox type="error"] Yeah message here! [/msgbox]
Info message [msgbox type="info"] Yeah message here! [/msgbox]
